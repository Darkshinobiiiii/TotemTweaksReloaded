## Features
#### _General Features_
- Changing totem size for handheld
- Changing the totem pop size
- Disable Equip Animation of the totem
#### _Totem Pop Animation_
- Option to Disable the pop animation
- Configurable Totem Pop Animation Speed
- Lock Rotation Position (Locks the position of the totem during rotation)
- Disable Rotations (Disable all totem rotations)
